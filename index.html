Portfolio

Full stack team project with help from data science students who filtered the comments from Hacker News (news.ycombinator.com) and attached a sentiment analysis score from 0 to 1 to each comment. Our front end team was tasked with displaying the data in a meaningful way which we did through dynamic graphs from the frontend library Ant-Design and built tools for users to search and sort the comments by username or comment score. Personally built the search tool, favorite comments feature, and edit user form with React Hooks, Context API, Ant-Design, and Styled Components.
